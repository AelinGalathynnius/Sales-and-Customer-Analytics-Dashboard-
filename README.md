## 📌 Overview

This project is an interactive Tableau dashboard created to analyze sales performance and customer behavior.

The dashboard helps track revenue trends, customer metrics, and sales performance through clear visualizations and filters. It was built to simulate real-world business reporting and decision-making.

---

## 🎯 Objectives

* Monitor sales performance over time
* Analyze customer behavior and purchasing trends
* Identify top-performing products or segments
* Support business decisions with clear visuals
* Improve reporting efficiency with interactive dashboards

---

## 🛠️ Tools Used

* Tableau
* Data Visualization
* Dashboard Design
* KPI Reporting
* Business Analysis

---

## 📈 Dashboard Features

### Sales Analysis

* Revenue trends over time
* Monthly / yearly performance tracking
* Sales comparisons across periods
* Key KPI monitoring

### Customer Analysis

* Customer activity trends
* Purchase behavior insights
* High-value customer identification
* Segment performance tracking

### Interactive Elements

* Dynamic filters
* Drill-down navigation
* Easy-to-read charts and visuals
* User-friendly dashboard layout
